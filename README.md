PubGPT is a tool that can extract lots of relevant informations inside PubMed papers in a very simple way:
- Abstract
- Full text (when available)
- Genes
- Diseases
- Associations between genes and diseases (powered by LLMs and LangChain)
- MeSH terms
- Other terms
